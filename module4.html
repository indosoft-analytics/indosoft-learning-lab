NumPy Fundamentals

NumPy is the core library for numerical computing in Python. It provides a high-performance multidimensional array object and tools for working with these arrays.

Key Advantage: NumPy arrays are 50x faster than Python lists for numerical operations due to fixed-type storage and vectorized operations.

Array Creation

Different ways to create NumPy arrays with examples.

From Python lists/tuples
Built-in array creation functions (zeros, ones, arange, etc.)
Random array generation
Special arrays (identity, diagonal, etc.)
Example Code:
import numpy as np

# Creating arrays from lists
arr1 = np.array([1, 2, 3, 4])          # 1D array
arr2 = np.array([[1, 2], [3, 4]])      # 2D array

# Special arrays
zeros = np.zeros((3, 3))               # 3x3 zeros
ones = np.ones((2, 4))                 # 2x4 ones
identity = np.eye(3)                   # 3x3 identity matrix
range_arr = np.arange(0, 10, 2)        # [0, 2, 4, 6, 8]

# Random arrays
random_arr = np.random.rand(2, 3)      # Uniform distribution
normal_arr = np.random.normal(0, 1, (3, 3))  # Normal distribution

print("1D Array:", arr1)
print("Identity Matrix:\n", identity)
print("Random Array:\n", random_arr)

Array Operations

Vectorized operations and array manipulation.

Vectorized arithmetic operations
Broadcasting rules
Array indexing and slicing
Reshaping and transposing
Example Code:
import numpy as np

# Vectorized operations
a = np.array([1, 2, 3])
b = np.array([4, 5, 6])

print("Addition:", a + b)       # [5 7 9]
print("Multiplication:", a * 2) # [2 4 6]
print("Dot product:", np.dot(a, b))  # 32

# Broadcasting example
matrix = np.array([[1, 2], [3, 4]])
scalar = 2
print("Matrix + scalar:\n", matrix + scalar)

# Advanced indexing
arr = np.arange(12).reshape(3, 4)
print("Original array:\n", arr)
print("First row:", arr[0])      # [0 1 2 3]
print("Second column:", arr[:, 1]) # [1 5 9]
print("Submatrix:\n", arr[1:3, 0:2])

Practical Application

Complete example analyzing temperature data.

import numpy as np
import matplotlib.pyplot as plt

# Generate synthetic temperature data (30 days)
np.random.seed(42)
days = np.arange(1, 31)
temps = 20 + 10 * np.sin(2 * np.pi * days / 30) + np.random.normal(0, 2, 30)

# Calculate statistics
mean_temp = np.mean(temps)
max_temp = np.max(temps)
min_temp = np.min(temps)
std_dev = np.std(temps)

# Find heat wave (temps > mean + 1.5*std)
heat_wave = days[temps > (mean_temp + 1.5 * std_dev)]

print(f"Average temperature: {mean_temp:.1f}°C")
print(f"Temperature range: {min_temp:.1f}°C to {max_temp:.1f}°C")
print(f"Heat wave days: {heat_wave}")

# Plot results
plt.figure(figsize=(10, 5))
plt.plot(days, temps, 'b-o', label='Daily Temp')
plt.axhline(mean_temp, color='r', linestyle='--', label='Mean Temp')
plt.fill_between(days, mean_temp-std_dev, mean_temp+std_dev, 
                color='gray', alpha=0.2, label='±1 Std Dev')
plt.scatter(heat_wave, temps[temps > (mean_temp + 1.5 * std_dev)], 
            color='red', s=100, label='Heat Wave')
plt.title('Monthly Temperature Analysis')
plt.xlabel('Day of Month')
plt.ylabel('Temperature (°C)')
plt.legend()
plt.grid(True)
plt.show()
Pro Tip: For large-scale numerical computations:
  • Use np.matmul() or @ for matrix multiplication
  • For very large arrays, consider memory-efficient operations
  • Explore scipy.linalg for advanced linear algebra
