Core Concepts

Data Structures

Series

One-dimensional labeled array capable of holding any data type

import pandas as pd
s = pd.Series([1, 3, 5, np.nan, 6, 8],
              index=['A', 'B', 'C', 'D', 'E', 'F'])
DataFrame

Two-dimensional labeled data structure with columns of potentially different types

data = {'Name': ['John', 'Anna', 'Peter'],
        'Age': [28, 24, 35]}
df = pd.DataFrame(data)

Data Input/Output

Method Description
read_csv() Read comma-separated values (csv) file
read_excel() Read Excel file
read_sql() Read SQL query or database table
to_csv() Write object to comma-separated values file

Data Cleaning

Handling Missing Data
# Detect missing values
df.isna()

# Drop rows with missing values
df.dropna()

# Fill missing values
df.fillna(value)
Data Transformation
# Remove duplicates
df.drop_duplicates()

# Replace values
df.replace({'old': 'new'})

# Rename columns
df.rename(columns={'old':'new'})

Hands-on Exercise

Customer Data Analysis

You've been given a dataset of customer transactions with some missing and inconsistent data. Your task is to clean and analyze this data.

Requirements:
  1. Load the provided customers.csv dataset
  2. Clean missing values (choose appropriate strategy)
  3. Calculate average purchase amount by customer segment
  4. Identify top 5 customers by total spending
  5. Save cleaned data to new CSV file
Pandas Cheat Sheet
Selection
df[col]          # Select column
df.loc[row]     # Select row by label
df.iloc[idx]    # Select row by position
Sorting
df.sort_values('col')
df.sort_index()
Aggregation
df.groupby('col').mean()
df.agg(['mean', 'sum'])
